Team Github Training


(0) First, get everyone signed up for a github account and make them a member of expertsystemsolutionsdevteam

Go to http://github.com

Sign up for github.com and email marty your username on github.


(1) Get a clone of the "Training" project onto your raspberry pi

> cd /var/www/html
> git clone https://github.com/expertsystemsolutionsdevteam/training.git training
Cloning into 'training'...
Username for 'https://github.com': expertsystemsolutionsdevteam
Password for 'https://expertsystemsolutionsdevteam@github.com': 
remote: Counting objects: 18, done.
remote: Compressing objects: 100% (14/14), done.
remote: Total 18 (delta 2), reused 13 (delta 0), pack-reused 0
Unpacking objects: 100% (18/18), done.
Checking connectivity... done.


(2) Add a file that is under the "Training" project

First you will cd down into the training directory. Then add a file using nano call it hello_world_{your name}.html. Put your name in the place of {your name}. Once inside of nano you can put the text "Hello Github!!" and save the file. 

> cd /var/www/html/training/github/hello_world
> nano hello_world_{your name}.html


(3) Now run the git status command

Notice that your file shows up as an untracked file

> cd /var/www/html/training/github/hello_world
> git status 
On branch master
Your branch is up-to-date with 'origin/master'.
Changes not staged for commit:
  (use "git add ..." to update what will be committed)
  (use "git checkout -- ..." to discard changes in working directory)

Untracked files:
  (use "git add ..." to include in what will be committed)

	hello_world_marty.html

no changes added to commit (use "git add" and/or "git commit -a")


(4) Now add the file to your local git instance using "git add {filename}" 

> cd /var/www/html/training/github/hello_world
> git add hello_world_marty.html


(5) Now run git status again

Notice that the file is no longer 'untracked'. It is under the "Changes to be commited"

> git status 
On branch master
Your branch is up-to-date with 'origin/master'.
Changes to be committed:
  (use "git reset HEAD ..." to unstage)

	new file:   hello_world_marty.html


(6) Now run 'git commit {filename}'

Note: This commits the file to your local copy of the repository

> cd /var/www/html/training/github/hello_world
> git commit -m "Commiting the hello world file with my name" hello_world_marty.html
 [master 6594353] hello_world_marty.html
 1 file changed, 3 insertions(+)
 create mode 100644 github/hello_world_marty.html


(7) Now run 'git push origin'

Note: This pushes your file up to the github repository

> git push origin
