Team Github Training


(1) Get a clone of the "Training" project onto your raspberry pi

> cd /var/www/html
> github clone https://github.com/expertsystemsolutionsdevteam/training.git training

(2) Add a file that is under the "Training" project

First you will cd down into the training directory. Then add a file using nano call it hello_world_{your name}.html. Put your name in the place of {your name}. Once inside of nano you can put the text "Hello Github!!" and save the file. 

> cd /var/www/html/training/
> nano hello_world_{your name}.html

(3) Now run the github status command

> cd /var/www/html/training/
> github status
